### bind9 vulnerability

A security issue affects these releases of Ubuntu and its derivatives:

* Ubuntu 15.04
* Ubuntu 14.04 LTS
* Ubuntu 12.04 LTS

### Summary

Bind could be made to crash if it received specially crafted network traffic.

### Software description

* bind9
