### gpgme1.0 vulnerability

A security issue affects these releases of Ubuntu and its derivatives:

* Ubuntu 14.04 LTS
* Ubuntu 12.04 LTS
* Ubuntu 10.04 LTS

### Summary

GPGME could be made to crash or run programs as your login if it processed a specially crafted certificate.

### Software description

* gpgme1.0 

### Details

Tom치코 Trnka discovered that GPGME incorrectly handled certain certificate line lengths. An attacker could use this issue to cause applications using GPGME to crash, resulting in a denial of service, or possibly execute arbitrary code. 

### Update instructions
